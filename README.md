# 基于SpringBoot的停车场管理系统 java846

## 项目概述

本项目是基于SpringBoot开发的停车场管理系统，实现前后端分离，具备车位信息管理、车辆进出管理、费用设置与缴纳等核心功能。

## 技术栈

- **前端技术**：Vue、Element-UI
- **后端技术**：SpringBoot、MyBatis Plus
- **数据库**：MySQL
- **开发语言**：Java

## 功能模块

### 系统角色

#### 管理员用户

- 用户信息管理
- 车位信息管理
- 车位费用设置
- 停泊车辆管理
- 车辆进出管理
- 登录日志查询

#### 车主用户

- 个人中心
- 预定停车位
- 缴费信息
  - 停车缴费记录
  - 待缴费
  - 账户余额充值

## 系统运行环境

- 前端：支持现代浏览器
- 后端：Java 1.8 或以上版本，SpringBoot 2.x
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 环境准备：确保Java、MySQL等环境已经安装且配置正确。
2. 数据库搭建：执行SQL脚本建立数据库和表结构。
3. 后端部署：配置好相关参数后，通过SpringBoot命令或容器化部署后端服务。
4. 前端部署：构建前端项目并部署到Web服务器。
